¿Qué es XML?

XML es un lenguaje de marcado que se utiliza para describir datos. XML fue diseñado para transportar y almacenar datos, y es particularmente adecuado para ser compartido entre sistemas y aplicaciones diferentes.

¿Qué es JSON?

JSON es un formato de texto ligero para el intercambio de datos. JSON es independiente del lenguaje y fácil de leer por los humanos. JSON es un subconjunto de la notación de objetos de JavaScript.

Diferencias y Semejanzas de XML y JSON

XML y JSON son dos formatos de datos diferentes que se utilizan para almacenar y transportar datos.

  • XML fue diseñado para almacenar y transportar datos, y es particularmente adecuado para ser compartido entre sistemas y aplicaciones diferentes.
  • JSON fue diseñado para ser un formato de intercambio de datos ligero. JSON es un subconjunto de la notación de objetos de JavaScript.

AJAX

AJAX es una técnica para crear aplicaciones web interactivas. AJAX permite que las páginas web se actualicen asincrónicamente mediante el intercambio de pequeños pedazos de datos entre el navegador y el servidor.
