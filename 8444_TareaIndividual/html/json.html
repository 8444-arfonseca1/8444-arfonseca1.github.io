¿Qué es JSON?

JSON, cuyas siglas significan en verdad JavaScript object notation que, en español se traducen como, notación de objetos de JavaScript, es un formato de intercambio de datos que resulta muy fácil de leer y escribir para los programadores y sencillo de interpretar y crear para las máquinas.
Es utilizado para transmitir datos en aplicaciones web (por ejemplo, enviar y recibir datos de una API) y se basa en el lenguaje de programación JavaScript. JSON utiliza una sintaxis de objetos de JavaScript para almacenar y transmitir datos de manera que sea fácil de leer y escribir para humanos y también fácil de interpretar y procesar por máquinas. JSON es un formato de texto plano, por lo que se puede transmitir a través de Internet y utilizar en una amplia variedad de lenguajes de programación.

Para qué sirve:

JSON se utiliza a menudo como alternativa a otras bases de datos y formatos de almacenamiento de datos, como XML o CSV. Es especialmente popular en aplicaciones web, donde se puede convertir fácilmente desde y hacia otros formatos como HTML y JavaScript. Además, como la sintaxis de JSON es sencilla y fácil de entender, muchos desarrolladores lo encuentran mucho más fácil de usar que otros formatos de datos.

Características:

  • Consiste en pares "clave - valor".
  • Requiere usar comillas dobles para las cadenas y los nombres de propiedades. Las comillas simples no son válidas.
  • Es un formato flexible, ligero y fácilmente transferible a través de las redes.
  • Los valores pueden cadenas, números o boleanos, así como otros objetos JSON, con cualquier nivel de anidación.
  • Encontrar elementos HTML por colecciones de objetos HTML.
