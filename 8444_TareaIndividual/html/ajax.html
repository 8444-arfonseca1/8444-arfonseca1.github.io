¿Qué es AJAX?

AJAX significa JavaScript asíncrono y XML (Asynchronous JavaScript and XML) se refiere a un grupo de tecnologías que se utilizan para desarrollar aplicaciones web. Al combinar estas tecnologías, las páginas web parece que son más receptivas puesto que los paquetes pequeños de datos se intercambian con el servidor y las páginas web no se vuelven a cargar cada vez que un usuario realiza un cambio de entrada. Ajax permite que un usuario de la aplicación web interactúe con una página web sin la interrupción que implica volver a cargar la página web. La interacción del sitio web ocurre rápidamente sólo con partes de la página de recarga y renovación.

AJAX se compone de las siguientes tecnologías:

  • XHTML y CSS para presentar información.
  • DOM.
  • El objeto XMLHttpRequest para manipular los datos de forma asíncrona con el servidor web.
  • XML, HTML y XSLTpara el intercambio y la manipulación de datos.
  • Se visualiza JavaScript para enlazar solicitudes e información de datos.

En una aplicación web Ajax, no se interrumpe el usuario en interacciones con la aplicación web. El motor de Ajax o el intérprete JavaScript permite que el usuario interactúe con la aplicación web independientemente del transporte HTTP procedente del servidor o que tenga el servidor como destino representando la interfaz y gestionando las comunicaciones con el servidor en nombre del usuario.



Características:

  • Asincronía: AJAX permite enviar y recibir datos del servidor de manera asíncrona, lo que significa que la página no se recarga completamente. Esto hace que las aplicaciones web sean más rápidas y sensibles al usuario.
  • Interacción en tiempo real: Con AJAX, las aplicaciones web pueden actualizar automáticamente su contenido sin necesidad de recargar la página completa. Esto permite una interacción en tiempo real con el servidor y mejora la experiencia del usuario.
  • Mejoras en la usabilidad: AJAX permite que las aplicaciones web sean más fáciles de usar al permitir la actualización de contenido sin necesidad de recargar la página completa.
  • Mejoras en el rendimiento: AJAX permite reducir la cantidad de datos que se envían y reciben entre el navegador y el servidor, lo que puede mejorar el rendimiento de la aplicación.
  • Independiente del lenguaje de programación del servidor: AJAX se puede utilizar con diferentes lenguajes de programación del servidor, como PHP, Java o .NET.
  • Soporte en la mayoría de los navegadores web: AJAX es compatible con la mayoría de los navegadores web modernos, lo que lo hace accesible para la mayoría de los usuarios.
